# OG Images

This folder should contain Open Graph images for social media sharing.

## Required Images

All images should be **1200x630px** PNG format.

| File | Content Description |
|------|---------------------|
| `og-home.png` | Logo + "AI Cost Tracking Proxy" + Dashboard mockup + Provider logos (OpenAI, Anthropic) |
| `og-docs.png` | Logo + "Documentation" + Code snippet visual |
| `og-sign-up.png` | Logo + "Start Free" + Feature highlights |
| `og-sign-in.png` | Logo + "Dashboard" + Simple UI preview |

## Design Specifications

- **Dimensions**: 1200x630px
- **Format**: PNG
- **Background**: Dark (#050505)
- **Accent Color**: Amber (#F59E0B)
- **Font**: Inter (matches the dashboard)

## Testing

After creating the images, test them with:
1. [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
2. [Twitter Card Validator](https://cards-dev.twitter.com/validator)
3. [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)
